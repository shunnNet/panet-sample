# panet (demo)
幫助表演藝術工作者與案主交流的平台，協助導演找到演員，演員可以找到導演。不再需要依靠人脈關係。

## Feature
- 會員系統、後台
- 透過「技能」等條件搜尋表演藝術人才
- 經營個人頁面，整合 Facebook 粉絲專頁
- 張貼工作機會、提出作品構想、上傳作品、張貼活動

## Tech
- Vue.js 及各項元件
- pug : SSR 後端動態渲染 
- SCSS + BEM
- node.js + express + mongoose + pug

`/public` : static files
`/views`  : pug templates
